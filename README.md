# TC4 Mobile (Expo SDK 54)

## Instalação
```bash
npm install
npx expo start
```
Edite `app.json > expo.extra.apiUrl` para apontar ao backend (Android emulador: `http://10.0.2.2:3000`).

## Telas
- Lista de posts (com busca), leitura do post.
- Login/Logout + autorização por perfil.
- Admin: CRUD de posts.
- Admin: CRUD paginado de docentes e alunos.
